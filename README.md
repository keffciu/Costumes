## System Kostiumów

Pozwala graczom na zmianę wyglądu bez wpływu na statystyki gry.

## Wymagania
- Wersja serwera Minecraft: 1.18.2 - 1.21.5
- Wymagane pluginy:
  - [PacketEvents](https://github.com/retrooper/packetevents)
  - [Vault](https://github.com/MilkBowl/Vault)
